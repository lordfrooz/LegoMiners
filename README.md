# Tempo Topia

Vercel-ready Next.js landing page inspired by the pacing and section flow of `tempo.xyz`, rebuilt as an original `Tempo Topia` brand experience.

## Requirements

- Node.js `>=20.9.0`
- npm

## Local development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

## Vercel

This project uses the standard Next.js app router and deploys to Vercel without extra configuration.
